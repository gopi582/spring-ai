As part of this project covered below by using open ai,

1. Chat Client example: http://localhost:8080/model/chat/explain me about java 21 features
2. Generating the image url based on the prompt: http://localhost:8080/model/image/generate an image like kid playing in basket ball court
3. Generating the text from image: http://localhost:8080/model/image-to-text (Sample image already placed in project src/main/resource)
4. Generating the Text from Audio: http://localhost:8080/model/audio-to-text (Sample Audio  already placed in project src/main/resource)
5. Generating the audio file based on the text passed as prompt:  http://localhost:8080/model/text-to-audio/i want to learn spring AI (Upon hitting this url an audio file will be downloaded)
